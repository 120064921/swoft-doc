# 应用组件
Swoft 采用组件化的开发模式，可自由组合各个组件来满足自己的业务实现，在简单业务下可以免于臃肿，在复杂业务下又游刃有余。  
以下为官方团队开发提供的组件，若有优秀的组件亦可联系 Swoft开发团队 增加在此列表上
